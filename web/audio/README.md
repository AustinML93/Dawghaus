# Fight song audio

Drop a file named **`fight-song.mp3`** in this folder to make the 🎺 Fight Song
button play the real recording of *Bow Down to Washington* (Lester J. Wilson, 1915 — public domain).

If no file is present, the button synthesizes a triumphant brass fanfare with the
Web Audio API instead, so it always does something — and it works offline.

`fight-song.mp3` is git-ignored on purpose (don't commit large/owned audio to the repo).

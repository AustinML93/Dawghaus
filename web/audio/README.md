# Audio files

Drop these files in this folder to use real recordings. Both are git-ignored on
purpose (don't commit large/owned audio to the repo), and both buttons fall back
to a synthesized sound if the file is missing, so they always do something —
and they work offline.

| File | Button | Fallback if missing |
|------|--------|---------------------|
| `fight-song.mp3` | 🎺 Fight Song (top bar) | synthesized brass fanfare |
| `touchdown.mp3`  | 🚨 Touchdown siren (floating button) | synthesized wailing siren |

*Bow Down to Washington* (Lester J. Wilson, 1915) is public domain.

Both buttons are toggles: tap to play, tap again to stop.
